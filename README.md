### `npm run-all -D'
express 서버를 실행하고, 다른 터미널을 열어서 react를 실행해야하지만, 
npm-run-all을 이용하면 하나의 명령어로 두개의 script를 실행 시킬 수 있다.
 - npm install npm-run-all -D
 - package.json scripts 수정
